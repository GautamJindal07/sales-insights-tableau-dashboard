## Problem Statement
This case study is based on a "Computer Hardware Business" operating in a dynamically changing market. The Sales Director needed real-time visibility into sales performance to support data-driven decisions.

## Solution
An interactive Sales Insights dashboard built in Tableau with a live MySQL connection via ODBC, providing real-time insights into revenue, regions, customers, and products.

## Key Features
- Revenue and Sales Quantity KPIs
- Region-wise and market-wise analysis
- Top customers and top products
- Month and Year based filtering
- Interactive Tableau dashboards

## Tools & Technologies
- Tableau Desktop / Tableau Public
- MySQL
- SQL
- ODBC Live Connection

## Data Preparation
- Currency conversion to INR
- Removal of blank and junk records
- Standardization of date and numeric fields
